**# WordPress-Inventory-App**

Create a simple inventory app inside you wordpress site. Doesn't require to have Woocomerce or eshop. It's a one page app that lets you track your inventory. It has some functionalitys like create different categories or tags for products. Also does have some statistics.


**# How to use**

Install plugin. Automatically a new section will be added in the admin panel. If you want to add it the front end for any reason create a page with the slug ../inventory-managment-app
